Необходимо разработать веб приложение - корпоративный справочник сотрудников.
Пользователи должны иметь возможности
- самостоятельная регистрация, логин, логаут.
- управление своим профилем. Профиль должен включать ФИО, должность, подразделение, контакты телефон, email, фотографию пользователя.
- просмотр списка сотрудников
- просмотр карточки сотрудника
- поиск сотрудников по фрагменту ФИО ,  номеру телефона, email

Используй стек технологий Python/Flask для backend, reactjs для frontend.

Для хранения данных используй Postgres
